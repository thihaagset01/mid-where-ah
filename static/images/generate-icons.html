MidWhereAh Icon Generator

How to save these icons:

  1. Right-click on each icon below
  2. Select "Save image as..."
  3. Save with the appropriate filename:
    • 192x192 icon → icon-192x192.png
    • 512x512 icon → icon-512x512.png
    • 48x48 icon → favicon.ico (or favicon.png and convert to .ico)
  4. Save all files in the static/images directory
